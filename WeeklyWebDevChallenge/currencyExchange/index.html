Stretch Goals

Since I've been learning and experimenting with APIs recently I decided to give it a shot this week.

I'm wondering if APIs and cloud database options like Google Firebase will replace conventional hosting one day. It's hard to figure out exactly what the cost would be though, with conventional hosting costs are fixed each month and you know what they are!

It seems that many API vendors are free up to a certain number of requests but then get costly after that. Same with cloud databases, I wouldn't want to get a $1500 bill one month because someone decided to make a game out of making repeated database requests!

The API I found was at https://exchangeratesapi.io. It was easier to use than pretty much any other api I've tried. Some have better docs that others!

Since getting data from an external API takes some time this also meant coming to grips with promises which is really not my forte, at least not yet.

It seems to work but the API doesn't return rate for some currencies (try converting to FJD which is Fijian currency) which results in a 0 amout, at least it did for me when testing it.

I'm thinking that the shipping cost to send a swag prize to Canada (in the unlikely event that I actually won) would be much more that the item itself! 🤣
